Setup Guide

Everything you need to deploy and manage the eSociety billing system for housing societies.

1

Initial Setup

One-time configuration by the system administrator

1

Create Google Apps Script Project

Navigate to script.google.com and sign in with your Google account. Click "New Project" to create a new Apps Script project.

2

Copy the Backend Code

Open google-apps-script.js from your project folder. Copy all the code and paste it into the Apps Script editor, replacing any existing code.

3

Deploy as Web App

Click Deploy → New deployment. Configure:

  • Type: Web app
  • Execute as: Me
  • Who has access: Anyone
4

Authorize Permissions

Click "Authorize access" and grant the required permissions. This allows the app to create spreadsheets in your Google Drive.

5

Copy Web App URL

After deployment, copy the generated URL:

https://script.google.com/macros/s/ABC...XYZ/exec
6

Configure Application

Open js/config.js and update the WEB_APP_URL with your copied URL:

// js/config.js const CONFIG = { WEB_APP_URL: 'YOUR_URL_HERE' };
7

Deploy to Web Hosting

Upload all files to your web hosting provider (GitHub Pages, Netlify, Vercel, or any web server). Your system is ready!

Setup Complete!
You can now login as Super Admin and start creating societies.
2

Creating a New Society

Register a new society in the system

1

Login as Super Admin

Open the application and select the "Super Admin" tab. Enter your credentials and sign in.

2

Access Dashboard

After login, you'll see the Super Admin Dashboard with all societies and statistics.

3

Click "Create Society"

Click the "+ Create Society" button in the Societies section.

4

Fill Society Details

  • Society Code: Unique 3-10 character identifier (e.g., SUNSHINE)
  • Society Name: Full name (e.g., "Sunshine Apartments")
  • Admin Password: Secure password (min 6 characters)
5

Save Credentials

After creation, securely save:

  • Society Code & Admin credentials
  • Google Spreadsheet link
  • Member Login Link (share with members)
Super Admin Credentials
Username and password are generated during initial setup. Check Google Apps Script logs after running setupSuperadminSheet().
3

Sharing Access with Members

Distribute login access to society members

1

Get Member Login Link

From Super Admin Dashboard, click the view icon next to any society. Click "Copy Link" to copy the Member Login Link.

2

Share the Link

Distribute via:

  • WhatsApp group
  • Email / SMS
  • QR code on notice board
3

Provide Login Credentials

Society admin creates member accounts from the Admin Dashboard. Share individual usernames and passwords with members.

4

Members Bookmark Link

Ask members to bookmark the link. The society is auto-selected from the URL - they only need username and password.

QR Code Tip
The system generates QR codes automatically! Download and print them for your notice board.
4

Bulk Import Data

Import buildings, flats, and members via CSV files

Import Order Required
1. Buildings → 2. Flats → 3. Members
Each step depends on the previous one existing.
1

Navigate to Import

Login as Society Admin → Master Data → Import Data tab.

2

Download Template

Click "Download Template" for the data type you want to import.

3

Fill Your Data

Open in Excel/Google Sheets, replace sample data with your data, save as CSV.

4

Upload & Import

Upload your CSV file. Preview shows validation - green = valid, red = errors. Click Import.

Frequently Asked Questions

Common questions and answers

Where is the society data stored?

Each society's data is stored in a separate Google Spreadsheet in the Super Admin's Google Drive. This ensures complete data isolation between societies.

Can I access data directly in Google Sheets?

Yes! From the Super Admin Dashboard, click the view icon for any society to get a direct link to its Google Spreadsheet.

What happens if I delete a society?

Deleting only deactivates the society - data is preserved in Google Sheets. Members can't login, but you can reactivate later if needed.

How many societies can I create?

No limit! Each society gets its own spreadsheet, so you can manage as many societies as needed.

Do members need to know the society code?

No. When members use the Member Login Link, the society is automatically selected. They only need their username and password.

Can I use this offline?

No, an internet connection is required as the data is stored in Google Sheets.

Need help? Contact your system administrator.
